# StoryCraft Studio

StoryCraft Studio is a powerful, AI-enhanced application designed for aspiring authors, screenwriters, and content creators. It provides a comprehensive suite of tools to help you brainstorm, outline, develop, and write your next masterpiece. From character creation and world-building to generating plot points with AI, StoryCraft Studio is your all-in-one creative partner.

## Features

-   **Project Dashboard**: Get a high-level overview of your project, including stats on characters, worlds, and manuscript sections.
-   **Story Templates**: Kickstart your writing process with classic narrative structures like the "Three-Act Structure" or genre-specific templates like "Murder Mystery."
-   **AI Outline Generator**: Simply provide a genre and a story idea, and let the AI generate a detailed, chapter-by-chapter outline for you.
-   **Character Builder**: Create and manage detailed character profiles. Use the AI assistant to generate rich backstories and motivations from a simple concept.
-   **World Building**: Construct the universe of your story. Define its history, geography, and unique systems, with AI assistance to flesh out your ideas.
-   **AI Writing Assistant**: Overcome writer's block by generating scene descriptions, character dialogue, or even alternative endings based on your prompts.
-   **Multi-language Support**: Fully localized UI available in English and German.
-   **Export Options**: Export your entire project—including characters, worlds, and the full manuscript—as a single Markdown or text file.

## Getting Started

### Prerequisites

This application runs entirely in the browser. You do not need to install any dependencies. A modern web browser like Chrome, Firefox, or Safari is all that's required.

### API Key Setup

StoryCraft Studio uses the Google Gemini API for its AI-powered features. To use these features, you must have a Google Gemini API key.

The application is configured to securely access your API key from the environment variable `process.env.API_KEY`. Please ensure this variable is set in the environment where you are running the application. **The application will not ask you to enter a key in the UI.**

## Usage

1.  **Dashboard**: Start here to see your project's progress. You can set your project title and logline.
2.  **Templates**: If you're starting a new project, go to the Templates view to choose a structure. Applying a template will populate your manuscript with predefined sections.
3.  **Outline Generator**: For a more custom structure, use the AI Outline Generator. Enter your genre and idea, and the AI will create a plot outline that you can apply to your manuscript.
4.  **Characters & World Building**: Use these sections to create the cast and setting of your story. The "Generate with AI" feature can quickly turn a simple idea into a detailed profile.
5.  **AI Writer**: Whenever you feel stuck, head to the AI Writer. Provide context, choose what you need (dialogue, description, etc.), and get instant creative input.
6.  **Writing**: Fill in the content for each section of your manuscript directly on the Dashboard.
7.  **Settings**: Change the application's display language between English and German.
8.  **Export**: Once you're ready, go to the Export view to download your complete work.

# StoryCraft Studio (Deutsch)

StoryCraft Studio ist eine leistungsstarke, KI-gestützte Anwendung für angehende Autoren, Drehbuchautoren und Content-Ersteller. Es bietet eine umfassende Sammlung von Werkzeugen, die Sie beim Brainstorming, Gliedern, Entwickeln und Schreiben Ihres nächsten Meisterwerks unterstützen. Von der Charaktererstellung und dem Weltenbau bis hin zur Generierung von Handlungspunkten mit KI ist StoryCraft Studio Ihr kreativer All-in-One-Partner.

## Funktionen

-   **Projekt-Dashboard**: Erhalten Sie einen Überblick über Ihr Projekt, einschließlich Statistiken zu Charakteren, Welten und Manuskriptabschnitten.
-   **Story-Vorlagen**: Starten Sie Ihren Schreibprozess mit klassischen Erzählstrukturen wie der "Drei-Akte-Struktur" oder genre-spezifischen Vorlagen wie "Krimi".
-   **KI-Gliederungsgenerator**: Geben Sie einfach ein Genre und eine Story-Idee an, und die KI generiert eine detaillierte, kapitelweise Gliederung für Sie.
-   **Charakter-Editor**: Erstellen und verwalten Sie detaillierte Charakterprofile. Nutzen Sie den KI-Assistenten, um aus einem einfachen Konzept reichhaltige Hintergrundgeschichten und Motivationen zu generieren.
-   **Weltenbau**: Erschaffen Sie das Universum Ihrer Geschichte. Definieren Sie Geschichte, Geografie und einzigartige Systeme, mit KI-Unterstützung zur Ausarbeitung Ihrer Ideen.
-   **KI-Schreibassistent**: Überwinden Sie Schreibblockaden, indem Sie Szenenbeschreibungen, Charakterdialoge oder sogar alternative Enden basierend auf Ihren Eingaben generieren.
-   **Mehrsprachige Unterstützung**: Vollständig lokalisierte Benutzeroberfläche in Englisch und Deutsch verfügbar.
-   **Exportoptionen**: Exportieren Sie Ihr gesamtes Projekt – einschließlich Charakteren, Welten und dem vollständigen Manuskript – als einzelne Markdown- oder Textdatei.

## Erste Schritte

### Voraussetzungen

Diese Anwendung läuft vollständig im Browser. Sie müssen keine Abhängigkeiten installieren. Ein moderner Webbrowser wie Chrome, Firefox oder Safari ist alles, was Sie benötigen.

### API-Schlüssel-Einrichtung

StoryCraft Studio verwendet die Google Gemini API für seine KI-gestützten Funktionen. Um diese Funktionen nutzen zu können, müssen Sie einen Google Gemini API-Schlüssel besitzen.

Die Anwendung ist so konfiguriert, dass sie sicher auf Ihren API-Schlüssel aus der Umgebungsvariable `process.env.API_KEY` zugreift. Bitte stellen Sie sicher, dass diese Variable in der Umgebung, in der Sie die Anwendung ausführen, gesetzt ist. **Die Anwendung wird Sie nicht auffordern, einen Schlüssel in der Benutzeroberfläche einzugeben.**

## Verwendung

1.  **Dashboard**: Beginnen Sie hier, um den Fortschritt Ihres Projekts zu sehen. Sie können Ihren Projekttitel und Ihre Logline festlegen.
2.  **Vorlagen**: Wenn Sie ein neues Projekt beginnen, gehen Sie zur Ansicht "Vorlagen", um eine Struktur auszuwählen. Das Anwenden einer Vorlage füllt Ihr Manuskript mit vordefinierten Abschnitten.
3.  **Gliederungsgenerator**: Für eine individuellere Struktur verwenden Sie den KI-Gliederungsgenerator. Geben Sie Ihr Genre und Ihre Idee ein, und die KI erstellt eine Handlungsskizze, die Sie auf Ihr Manuskript anwenden können.
4.  **Charaktere & Weltenbau**: Nutzen Sie diese Bereiche, um die Besetzung und das Setting Ihrer Geschichte zu erstellen. Die Funktion "Mit KI generieren" kann eine einfache Idee schnell in ein detailliertes Profil verwandeln.
5.  **KI-Schreiber**: Wenn Sie einmal nicht weiterwissen, gehen Sie zum KI-Schreiber. Geben Sie Kontext an, wählen Sie aus, was Sie benötigen (Dialog, Beschreibung usw.), und erhalten Sie sofort kreativen Input.
6.  **Schreiben**: Füllen Sie den Inhalt für jeden Abschnitt Ihres Manuskripts direkt im Dashboard aus.
7.  **Einstellungen**: Ändern Sie die Anzeigesprache der Anwendung zwischen Englisch und Deutsch.
8.  **Export**: Sobald Sie fertig sind, gehen Sie zur Exportansicht, um Ihre vollständige Arbeit herunterzuladen.
